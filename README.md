# FlaskApp

Simple CRUD web app built with Python, Flask and SQLite.

## Technologies
- Python 3
- Flask
- SQLite3
- HTML/CSS

## Features
- Add skills
- Edit skills
- Delete skills
- Data stored in SQLite database

## How to run
1. Install dependencies: pip install flask
2. Run: python app.py
3. Open: http://127.0.0.1:5000
